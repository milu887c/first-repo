# **¿HOW DO I PLAY?:**
Simple, you have to choose a move, Paper, Rock or Scissors.  
In the screen you can see that you can choose one with a number (1. Rock, 2. Paper and 3. Scissors).  
When you finally choose one the CPU will randomly pick  one option too.
